# cloud storage

```js

# create a new bucket
gsutil mb gs://<BUCKET_NAME>

# copy a file to a bucket
gsutil cp [MY_FILE] gs://[BUCKET_NAME]

```

## compute engine

```js

# list available regions
gcloud compute regions list

```
